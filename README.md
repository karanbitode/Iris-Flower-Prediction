# Iris-Flower-Prediction
Machine Learning Python Script implementing K-means clustering.

Dataset Columns :- 
1. sepal length
2. sepal width
3. petal length
4. petal width
5. class - Iris-setosa, Iris-versicolor or Iris-virginica
